## 链表 -- LinkedList

#### 优点
插入、删除操作只需改变前/后节点的next指向即可，时间复杂度为 O(1)

#### 缺点
查找一个节点需要遍历链表，时间复杂度为 O(n)

#### 时间复杂度 Time complexity

- prepend O(1)

- append O(1)

- lookup O(n)

- insert O(1)

- delete O(1)

#### 空间复杂度 Spatial complexity

- 链表的空间复杂度是 O(n)

#### 经典场景

- 链表使用的经典场景，LRU（least recently userd），最少最近使用策略
